# PWA Manual Test Checklist — Module 16

Выполнить перед каждым релизом. Отмечать дату и инициалы тестировщика.

## Установка и запуск

- [ ] Install prompt появляется на Android Chrome при первом заходе
- [ ] «Добавить на экран Домой» на iOS Safari (Share → Add to Home Screen)
- [ ] После установки приложение открывается в standalone-режиме (без адресной строки)
- [ ] Иконки 192×192 и 512×512 отображаются корректно
- [ ] Maskable-иконка отображается без обрезки на Android
- [ ] Splash screen (iOS) показывает логотип при запуске
- [ ] Shortcuts работают при долгом нажатии на иконку (Android 8+):
  - Журнал
  - Фото
  - Дефект

## Офлайн-режим

- [ ] Главная страница (/mobile) открывается офлайн из кэша
- [ ] Навигация между кэшированными страницами работает офлайн
- [ ] Некэшированный маршрут показывает страницу-заглушку (~offline)
- [ ] Создание записи ОЖР офлайн → попадает в sync-queue (видна в профиле)
- [ ] При восстановлении соединения sync-queue разгребается автоматически
- [ ] Иконка/счётчик sync-queue обновляется после синхронизации

## Push-уведомления

- [ ] При первом входе появляется запрос на разрешение уведомлений
- [ ] Уведомление приходит за < 5 секунд после события (согласование, дефект)
- [ ] Клик по уведомлению открывает нужный URL в приложении
- [ ] Отписка от уведомлений через настройки браузера/ОС работает корректно

## Камера и геолокация

- [ ] Кнопка камеры открывает нативный выбор (камера / галерея)
- [ ] Фото сжимается до < 1.5 МБ перед сохранением
- [ ] GPS-координаты прикрепляются к фото (широта/долгота/точность)
- [ ] Фото сохраняется офлайн в IndexedDB (видно в DevTools → Application → IndexedDB)

## Геозоны (SignWithGps)

- [ ] Подписание в пределах геозоны объекта — без предупреждений
- [ ] Подписание вне геозоны — показывает предупреждение с расстоянием
- [ ] Подписание вне геозоны всё равно возможно (с аудит-записью)

## Голосовой ввод

- [ ] Кнопка микрофона в форме записи ОЖР работает
- [ ] Распознаётся русская речь (ru-RU)
- [ ] Транскрипт добавляется в поле описания
- [ ] При недоступности Web Speech API кнопка скрыта/задизейблена

## Производительность

- [ ] Lighthouse PWA score ≥ 90
- [ ] Lighthouse Performance score ≥ 70
- [ ] First Contentful Paint < 3 сек на 3G (Lighthouse throttling)
- [ ] Storage quota < 80% (проверить в DevTools → Application → Storage)

## Безопасность

- [ ] VAPID private key не виден в client bundle (DevTools → Sources)
- [ ] IndexedDB не содержит пароли или JWT-токены
- [ ] При выходе (signOut) IndexedDB очищается
